
# 스마트폰 과의존 분석: 코로나 전후 변화 양상 탐구 (Streamlit Project)

# 1. 프로젝트 설명
본 프로젝트는 코로나19 팬데믹을 전후로 하여 스마트폰 과의존 양상이 어떻게 변화했는지 데이터 분석을 통해 탐구하고, 그 결과를 Streamlit을 활용하여 시각화 및 대화형 웹 애플리케이션으로 구현한 것입니다. 단순히 과의존율의 변화를 넘어, 사회 경제적 지표(실업률) 및 사용자 그룹별(일반/과의존) 선호 매체 차이까지 심층적으로 분석했습니다.

# 2. 주제
코로나 시기 전후 스마트폰 과의존 양상 변화 분석 및 비교 시각화

# 3. 주제 선택 이유
코로나19 팬데믹은 비대면 활동의 증가를 가져왔고, 이는 필연적으로 스마트폰 사용 시간 및 의존도 증가에 큰 영향을 미쳤습니다. 본 프로젝트는 이 시기를 기점으로 하여 스마트폰 과의존의 패턴 변화를 구체적인 데이터로 확인하고, 실업률 등 사회적 요인과의 연관성을 탐색하며, 사용자 그룹(일반 vs. 과의존) 간의 사용 콘텐츠 선호도 차이를 파악하여 스마트폰 중독 예방 및 대책 마련을 위한 객관적인 기초 자료를 제공하고자 했습니다.

# 4. 데이터 분석 내용
코로나 전후 과의존율 및 실업률 비교:코로나 팬데믹 기간(2020년 이후)을 전후로 한 스마트폰 과의존 위험군 비율 변화를 시각화했습니다.동일 기간의 전국 실업률 데이터를 함께 분석하여 사회 경제적 환경 변화와 스마트폰 의존도의 잠재적 연관성을 탐색했습니다.과의존 사용자 vs. 일반 사용자 의존 매체 차이:일반 사용자와 과의존 위험군 사용자가 스마트폰으로 주로 사용하는 **매체(예: 게임, SNS, 인터넷 검색, 동영상 시청)**의 선호도와 비중 차이를 비교 분석했습니다.이를 통해 과의존을 유발하는 주요 콘텐츠 유형을 파악하고, 사용자 그룹별 사용 패턴의 특성을 명확히 구분했습니다.
